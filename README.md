🛍️ Customer Shopping Behavior Analysis








📌 Overview

This project analyzes customer shopping behavior using 3,900+ transactions to identify patterns in spending, product preferences, and customer segments.
The goal is to generate actionable business insights using an end-to-end data analytics workflow.

⚙️ Tech Stack
🐍 Python (Pandas, NumPy) – Data Cleaning & Preprocessing
🛢️ MySQL – Business Query Analysis
📊 Power BI – Interactive Dashboard
🔍 Key Analysis
Customer segmentation (New, Returning, Loyal)
Revenue analysis by gender & age group
Top-rated & most purchased products
Discount usage vs spending behavior
Subscriber vs non-subscriber comparison
📈 Key Insights
Loyal customers contribute the highest revenue
Discount users can still be high-value customers
Subscription increases engagement and repeat purchases
Certain products are highly discount-dependent
💡 Business Impact
Helps optimize marketing strategies
Supports customer retention programs
Improves product positioning & pricing decisions
🚀 Workflow
Data Collection → Python (Cleaning & Feature Engineering) → 
MySQL (SQL Analysis) → Power BI (Visualization)
